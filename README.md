This is a [Next.js 14.2](https://nextjs.org/) project built using Typescript and Tailwind.

## Getting Started

### Requirements:

```bash
node version >= 22.0.0
```

### Setup:

```bash
yarn install
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Testing:

```bash
yarn test
```
